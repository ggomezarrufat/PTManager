## 🚀 **Cómo Usar la Aplicación:**

### **1. Iniciar la Aplicación:**
```bash
cd pt-manager-mobile
npx expo start --ios
```

### **2. En el Simulador:**
- La aplicación se abre automáticamente
- Navega entre pantallas usando los tabs
- Prueba todas las funcionalidades

### **3. Comandos Útiles:**
- `r` - Recargar aplicación
- `d` - Abrir debugger
- `j` - Debugger JS
- `m` - Mostrar menú
